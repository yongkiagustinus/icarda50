
# ICARDA Landing Page

A simple and modular HTML landing page project using SASS/SCSS for styling.
This structure allows clean component organization and easy styling maintenance.




## Run Locally

Clone the project

```bash
  git clone https://github.com/yongkiagustinus/icarda50.git
```

Go to the project directory

```bash
  cd icarda50
```

Install dependencies

```bash
  npm install
```

Compile SASS

```bash
  npm run sass
```
Watch SASS

```bash
  npm run sass:watch
```
